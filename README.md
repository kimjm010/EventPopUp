
## Description
* 이벤트 팝업창을 표시하는 기능입니다.
* 작업 기간: 2022.09.14 - 2022.09.16(2일)

## Environment
* 개발 환경(IDE): Xcode 13 ~ 14 버전
* 개발 언어: Swift, RxSwift
* Target: iOS 13.0 이상
* 소스 관리: Git

## Library
* PopupDialog

## Implementations
* 팝업 창 바깥 화면을 탭하면 팝업창이 닫힘
* 다시 보지 않기 버튼 탭한 경우 앱을 껏다 켜도 팝업창이 보이지 않음
* 자세히보기 버튼 탭한 경우 새로운 화면을 present방식으로 보여줌
